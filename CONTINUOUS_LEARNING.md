# سیستم یادگیری پیوسته
# Continuous Learning System

## 🎯 چیست؟
## What is it?

یک سیستم که **همیشه** در حال یادگیری است:
- مدل را train می‌کند
- از مدل نمونه‌های جدید تولید می‌کند
- نمونه‌های جدید را به dataset اضافه می‌کند
- دوباره train می‌کند
- این چرخه **بی‌نهایت** ادامه پیدا می‌کند

## 🚀 استفاده
## Usage

```bash
python continuous_train.py
```

**نکته**: این اسکریپت برای سرورهای 24/7 طراحی شده است.

## 🔄 چرخه یادگیری
## Learning Cycle

```
┌─────────────────────────────────┐
│  1. بارگذاری Dataset             │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  2. بارگذاری Model              │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  3. تولید 100 نمونه جدید       │
│     از Model                    │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  4. اضافه کردن به Dataset      │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  5. Train Model                 │
│     (2 epochs)                  │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  6. ذخیره Model                 │
└──────────────┬──────────────────┘
               ↓
┌─────────────────────────────────┐
│  7. انتظار 5 دقیقه              │
└──────────────┬──────────────────┘
               ↓
         (بازگشت به 1)
```

## ⚙️ تنظیمات
## Settings

در `continuous_train.py` می‌توانید تنظیمات را تغییر دهید:

```python
GENERATE_NEW_EXAMPLES = 100  # تعداد نمونه‌های جدید در هر چرخه
MIN_CYCLE_INTERVAL = 300     # فاصله بین چرخه‌ها (ثانیه) = 5 دقیقه
MAX_DATASET_SIZE = 10000     # حداکثر اندازه dataset
```

### توصیه‌ها:
- **GENERATE_NEW_EXAMPLES**: 50-200 (بیشتر = بهتر اما کندتر)
- **MIN_CYCLE_INTERVAL**: 300-1800 (5-30 دقیقه)
- **MAX_DATASET_SIZE**: 5000-20000 (بیشتر = بهتر اما کندتر)

## 📊 پیشرفت
## Progress

سیستم در هر چرخه:
- ✅ تعداد نمونه‌های تولید شده را نشان می‌دهد
- ✅ تعداد نمونه‌های dataset را نشان می‌دهد
- ✅ پیشرفت training را نشان می‌دهد
- ✅ زمان هر چرخه را نشان می‌دهد

## 💡 نکات مهم
## Important Tips

1. **اجرای 24/7**: این اسکریپت برای اجرای مداوم طراحی شده
2. **GPU لازم**: نیاز به GPU با حداقل 6GB VRAM
3. **فضای دیسک**: dataset و checkpoint ها فضا می‌گیرند
4. **پشتیبان‌گیری**: قبل از هر train، backup خودکار می‌شود
5. **توقف**: Ctrl+C برای توقف ایمن

## 🔧 مثال استفاده

```bash
# اجرا در background (Linux)
nohup python continuous_train.py > train.log 2>&1 &

# مشاهده لاگ
tail -f train.log

# توقف
pkill -f continuous_train.py
```
