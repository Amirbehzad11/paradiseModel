# سیستم آموزش تدریجی - Incremental Training

## نحوه کار
## How It Works

سیستم `train_incremental.py` به این صورت کار می‌کند:

1. **اولین بار**: دیتاست فعلی را لود می‌کند و train می‌کند
2. **بارهای بعدی**: 
   - اگر مدل از قبل وجود داشته باشد، از آن استفاده می‌کند
   - نمونه‌های جدید تولید می‌کند
   - آنها را به دیتاست اضافه می‌کند
   - با دیتاست بزرگتر train می‌کند

## استفاده
## Usage

```bash
# اولین بار (یا بعد از حذف final_model)
python train_incremental.py

# بارهای بعدی - هر بار دیتاست بزرگتر می‌شود
python train_incremental.py
```

## Benefits

- ✅ دیتاست خودش بزرگ می‌شود
- ✅ مدل هر بار بهتر می‌شود

- ✅ پشتیبان‌گیری خودکار از دیتاست
